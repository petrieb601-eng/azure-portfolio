📚 About Hash Functions

MD5

128-bit hash. Fast but not cryptographically secure. Use only for non-security checksums.

SHA-1

160-bit hash. Deprecated for security. Still used for Git commits and legacy systems.

SHA-256

256-bit hash. Industry standard. Secure and widely used for file verification.

SHA-512

512-bit hash. Most secure. Slower but provides maximum collision resistance.

Common Use Cases:

  • Download Verification: Verify downloaded files haven't been tampered with
  • File Integrity: Detect file corruption or unauthorized changes
  • Software Distribution: Publishers provide hashes to verify authentic downloads
  • Backup Verification: Ensure backup files match originals
